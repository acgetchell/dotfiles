# Computational Geometry Credit

Load this reference only for computational geometry, triangulation, mesh/topology, spatial-index, or robust-predicate citation work.

Map implemented or advertised techniques to appropriate primary sources and relevant implementation context. Check, when present:

- Delaunay/Voronoi theory, lifted-paraboloid duality, and incremental construction
- convex hull construction, visibility, conflict graphs, and point-in-polytope methods
- robust predicates, adaptive/exact arithmetic, floating-point filters, symbolic perturbation, and Simulation of Simplicity
- flips, Pachner/bistellar moves, local repair, and PL-manifold or topology validation
- point location, walking algorithms, spatial ordering, Hilbert/Morton curves, trees, and spatial indexes
- periodic or toroidal triangulations, quotient constructions, and image-point methods
- mesh and triangulation data structures, compact storage, handles/keys, secondary indexes, and cache schemes when externally inspired
- circumcenters, circumspheres, simplex volume, Gram determinants, quality metrics, and high-dimensional bounds
- benchmark datasets, methodology, reproducibility, and research-software practice when claimed

Do not assume a broad computational-geometry textbook supplies adequate credit for a named algorithm, robustness technique, or data structure with a known primary source. Conversely, do not add ornamental citations where the repository makes no borrowed-method or scientific claim.

Check local placement as well as bibliography presence: source/module docs or the nearest scientific topic should identify the method or point to the bibliography so readers can connect implementation to credit.
